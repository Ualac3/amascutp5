# Amascut P5

A small React + TypeScript app that counts occurrences of `shield us from your shadow...`, labels each hit as `BLOCK` or `MISS`, and recommends the next defensive.

## Run locally

1. `cd c:/Users/Micha/amascutp5`
2. `npm install`
3. `npm start`

## Project structure

- `src/App.tsx` — main UI and counter logic
- `src/phraseDetection.ts` — phrase detection and defensive suggestion rules
- `public/index.html` — app shell
